# Outputs Compartilhados

> Resultados e entregáveis de todos os agentes

---

## Estrutura

```
shared/outputs/
├── YYYY-MM-DD/          ← Pasta por data
│   ├── alfred/          ← Output do Alfred
│   └── [agente]/        ← Output de outros agentes
└── index.md             ← Índice de outputs importantes
```

---

## Como Usar

Cada agente salva seus resultados aqui para outros agentes acessarem.

**Alfred (principal):** Coordena e consolida outputs de todos.
