# Wolf Knowledge Base — Base de Conhecimento Validada

> Conhecimento curado pela Wolf Agency. Tudo aqui foi testado, validado e aprovado por Netto.
> O Gabi carrega automaticamente o conteúdo relevante a cada análise.

---

## Estrutura

```
knowledge-base/
├── playbooks/       ← Processos passo a passo (como fazer X)
├── frameworks/      ← Modelos de raciocínio e estrutura
├── benchmarks/      ← Números de referência por nicho/plataforma
└── case-studies/    ← Cases reais: o que funcionou + resultado
```

---

## Como adicionar conhecimento

**Formato recomendado para cada arquivo:**

```markdown
# [TÍTULO DO CONHECIMENTO]
# Validado por: Netto | Data: YYYY-MM
# Contexto: [quando usar / para quem]

## O que é
[1-2 parágrafos]

## Quando aplicar
- Situação A
- Situação B

## Como aplicar
[passo a passo ou framework]

## Resultado esperado
[o que muda / benchmark de sucesso]

## Erros comuns
- Erro X → como evitar
```

---

## Prioridade de conteúdo (o que Gabi mais precisa)

1. **Playbooks de escala** — CBO, ABO, como escalar sem quebrar aprendizado
2. **Criativos que converteram** — hook, ângulo, formato + resultado real
3. **Benchmarks por nicho** — CPM, CTR, CPA, ROAS por segmento
4. **Copy validada** — headlines, CTAs que funcionaram com dado
5. **Diagnósticos resolvidos** — "campanha X com problema Y → solução Z"
