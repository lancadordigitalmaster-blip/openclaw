# Lições Compartilhadas

> Aprendizados, erros e insights de todo o time de agentes

---

## Estrutura

```
shared/lessons/
├── README.md            ← Este arquivo
├── YYYY-MM-DD.md        ← Lições por data
└── by-topic/            ← Lições por tópico
    ├── comunicacao.md
    ├── automacoes.md
    └── [topico].md
```

---

## Tipos de Lição

| Tipo | Descrição | Exemplo |
|------|-----------|---------|
| ✅ Acerto | O que funcionou bem | "Checklist de qualidade reduziu retrabalho em 50%" |
| ⚠️ Erro | O que deu errado | "Cron muito frequente gerou spam" |
| 💡 Insight | Descoberta importante | "Netto prefere bullets a parágrafos longos" |
| 🔧 Ajuste | Mudança de processo | "Mudar horário do digest de 9h para 8h" |

---

## Como Registrar

Quando um agente aprender algo relevante:
1. Criar entrada em `YYYY-MM-DD.md`
2. Classificar por tipo
3. Taggear com agente responsável
