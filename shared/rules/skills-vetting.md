# PROTOCOLO DE AVALIACAO DE SKILLS DE TERCEIROS
# Aplicar a toda skill do ClawHub ou fonte externa antes de instalar

## Passos

1. Ler o codigo completo da skill antes de instalar (sem excecao)
2. Verificar quais permissoes a skill solicita. Rejeitar acesso nao justificado
3. Se acessa dados de clientes ou credenciais -> aprovacao obrigatoria de Netto
4. Testar em ambiente isolado (sem conectar a contas reais) na primeira execucao
5. Registrar em memory/skills-installed.md: nome, fonte, data, permissoes, resultado
6. Skills criadas on-the-fly pelo proprio Alfred nao precisam deste protocolo
