## Fórmula de Performance (15–60s)

```
HOOK (0–2s) → CONTEXTO (2–6s) → PROVA (6–20s) → PONTOS (20–45s) → CTA
```

---

## Bloco 1: Hook (0–2s)

**Objetivo:** Parar o dedo no scroll. Zero enrolação.

**Técnicas:**
- Texto grande na tela (pergunta provocativa)
- Movimento abrupto (zoom, transição)
- Expressão facial intensa
- Som/beat drop
- Contrarian statement

**Checklist:**
- [ ] Movimento nos primeiros 2s
- [ ] Texto ou fala começa imediatamente
- [ ] Sem introduções ("Oi pessoal...")

---

## Bloco 2: Contexto Rápido (2–6s)

**Objetivo:** Estabelecer do que se trata.

**Técnicas:**
- Uma frase de contexto
- Visual que reforça o tema
- Transição rápida

**Checklist:**
- [ ] Contexto em 3s ou menos
- [ ] Transição para prova imediata

---

## Bloco 3: Prova/Benefício (6–20s)

**Objetivo:** Mostrar que funciona ou vale a pena.

**Técnicas:**
- Antes/depois rápido
- Resultado específico ("Ganhei X em Y dias")
- Reação genuína
- Print de resultado

**Checklist:**
- [ ] Prova visual ou numérica
- [ ] Credibilidade estabelecida

---

## Bloco 4: Pontos-Chave (20–45s)

**Objetivo:** Entregar valor ou argumentos.

**Técnicas:**
- 2–3 pontos máximo
- Pattern interrupts entre pontos
- Texto na tela reforçando
- Cortes rápidos (ritmo acelerado)

**Pattern Interrupts:**
- Mudança de ângulo
- Zoom in/out
- Insert de imagem/vídeo
- Efeito sonoro
- Mudança de expressão

**Checklist:**
- [ ] Máximo 3 pontos
- [ ] Interrupt a cada 5–8s
- [ ] Texto destacando palavras-chave

---

## Bloco 5: CTA (Final)

**Objetivo:** Uma ação clara.

**Técnicas:**
- "Salva isso"
- "Comenta X"
- "Link na bio"
- "Me segue para mais"

**Checklist:**
- [ ] CTA único
- [ ] Texto na tela do CTA
- [ ] Loop natural (quando possível)

## Especificações Técnicas

### Safe Areas (Zona de Segurança)

```
┌─────────────────────────────┐
│     [NÃO COLOCAR NADA]      │  ← 250px topo
│  ┌───────────────────────┐  │
│  │                       │  │
│  │    CONTEÚDO SEGURO    │  │  ← Área segura
│  │                       │  │
│  └───────────────────────┘  │
│     [NÃO COLOCAR NADA]      │  ← 250px base
└─────────────────────────────┘
        (CTA do Instagram)
```

**Regras:**
- Texto principal: centro da tela
- Nada importante nos 250px do topo
- Nada importante nos 250px da base
- Legendas: dentro da área segura

### Legendas

**Formato:**
- Máximo 2 linhas
- Quebrar em ideias curtas
- Palavras-chave destacadas (cor diferente ou negrito)
- Stroke/sombra para contraste

**Modos:**
| Modo | Uso | Exemplo |
|------|-----|---------|
| Word-by-word | Alta energia, retenção | "Você" [quebra] "precisa" [quebra] "ver" [quebra] "isso" |
| Blocos | Premium, informativo | "3 dicas para vender mais" |
| Híbrido | Padrão Wolf | Bloco + destaque de palavras-chave |

### Ritmo e Cortes

**Densidade:**
- Reels: 1 corte a cada 2–4s
- Stories: 1 corte a cada 3–5s

**Variação de Crop:**
- Close-up (rosto)
- Medium (cintura pra cima)
- Wide (ambiente)
- Insert (prints, vídeos)

**Transições:**
- Hard cut (padrão)
- Zoom in/out
- Slide rápido
- Match cut

## Estilos por Plataforma

### Instagram Reels
- Ritmo acelerado
- Música trending (quando relevante)
- Hook visual forte
- CTA: salvamento e compartilhamento

### Stories
- Ritmo mais natural
- Interatividade (enquetes, perguntas)
- Sequência de 3–5 stories
- Swipe up/link (quando disponível)

### TikTok
- Hook nos primeiros 1s
- Sons trending
- Participação em trends (quando alinhado)
- Texto grande na tela

### YouTube Shorts
- Similar ao Reels
- Foco em retenção (vídeo precisa ser assistido)
- Thumbnail otimizado

---

## Checklist Final de Reels/Stories

- [ ] Hook nos primeiros 2s
- [ ] Contexto em até 6s
- [ ] Prova visual ou numérica
- [ ] Máximo 3 pontos
- [ ] Pattern interrupts frequentes
- [ ] Legendas em todas as falas
- [ ] Safe areas respeitadas
- [ ] CTA único e claro
- [ ] Loop natural (quando possível)
- [ ] Áudio limpo
- [ ] 1º frame forte (thumbnail)
- [ ] Música/SFX apropriados
