# Posicionamento, Naming, Tagline e Tom de Voz

Posicionamento não é o que você diz sobre si mesmo.
É o que a pessoa pensa quando ouve seu nome — antes de você falar qualquer coisa.
Se você precisa explicar seu posicionamento, ele ainda não existe.

---

## Posicionamento Diferenciado — Framework

### A pergunta central
"Quando [avatar] pensa em [categoria], pensa em quem?"
Se a resposta não for você: você tem presença, não posicionamento.

### Processo em 4 passos

```
PASSO 1 — MAPEIE O CAMPO
Para cada concorrente principal:
├── Promessa principal de comunicação
├── Avatar declarado
├── Tom de voz
└── O que NENHUM deles está dizendo (gap)

PASSO 2 — ENCONTRE O TERRITÓRIO VAZIO
Posicionamento forte ocupa espaço que:
├── Tem demanda real (o avatar quer isso)
├── Está subatendido (concorrentes não estão lá)
└── Você pode defender com credibilidade (você realmente entrega)

PASSO 3 — DEFINA O TERRITÓRIO EM UMA FRASE
"Eu sou o [único/melhor/mais específico] para [avatar] que [situação]"

Exemplos:
├── "A única agência de tráfego que atende só clínicas odontológicas"
├── "O método de copy para quem odeia escrever mas precisa vender"
└── "Gestão financeira para autônomos que ganham bem mas não sobra"

PASSO 4 — TESTE DE CLAREZA
Mostre para 5 pessoas do avatar sem contexto adicional.
Entenderam imediatamente para quem é e o que entrega?
Se hesitaram: precisa de mais especificidade.
```

## Naming — Criação de Nome

### Critérios obrigatórios antes de apresentar qualquer nome
```
FUNCIONAL:
├── Fácil de pronunciar em português
├── Fácil de lembrar após ouvir uma vez
├── Fácil de pesquisar (sem homônimos problemáticos)
└── Domínio e @ disponíveis (verifique ANTES de apresentar)

ESTRATÉGICO:
├── Sugere o benefício ou posicionamento sem ser literal demais
├── Tem personalidade (não genérico como "Marketing Pro")
├── Escala com o negócio (não prende a nicho estreito demais)
└── Funciona ao lado de: "agência", "método", "sistema", "escola"

EMOCIONAL:
├── Cria imagem mental imediata
├── O avatar quer ser associado a esse nome?
└── Tem algum nível de aspiração ou tensão
```

### Técnicas de geração

```
RESULTADO PERSONIFICADO
O que o produto transforma a pessoa em?
→ Wolf, Titan, Apex, Gabi, Vortex, Atlas

MECANISMO NOMEADO
Dê nome ao processo único:
→ "Método Ignição", "Protocolo Dark", "Sistema Foguete", "Squad Wolf Pack"

METÁFORA DO BENEFÍCIO
Imagem que representa o resultado:
→ "Alavanca" (crescimento), "Âncora" (estabilidade), "Farol" (clareza)

COMBINAÇÃO DE RADICAIS
Crie palavras novas com radicais que remetem ao benefício:
→ "Converta", "Traxion", "Adspark", "Scalify"

NOME + RESULTADO (personal brand)
→ "João Vendas", "Ana Copy", "Pedro Escala"
```

## Tagline — A Promessa Destilada

A tagline é a oferta em sua forma mais concentrada.
Se precisar de 3 frases para explicar a tagline: ela não é uma tagline.

### Fórmulas

```
RESULTADO DIRETO:
"[Verbo] + [resultado específico]"
→ "Escale sem perder margem."
→ "Venda enquanto dorme."

PARA QUEM + O QUÊ:
"Para [quem] que [resultado]"
→ "Para quem quer vender sem parecer vendedor."

CONTRASTE:
"Menos [indesejado]. Mais [desejado]."
→ "Menos achismo. Mais resultado."
→ "Menos ruído. Mais venda."

FILOSOFIA:
Captura a visão de mundo da marca.
→ "Dado primeiro. Criativo depois."
→ "O mercado recompensa quem entende de gente."

PROVOCAÇÃO (divide e atrai os certos):
→ "Marketing sem dado é esperança."
→ "Conteúdo sem estratégia é hobby."
```

## Tom de Voz — SOUL.md por Cliente

Documento que deve ser criado para cada cliente e lido antes de qualquer criação.

### Estrutura do SOUL.md

```
VOCABULÁRIO FREQUENTE:
├── 20 palavras/expressões que o cliente repete naturalmente
├── Gírias e bordões que identificam o estilo
└── Palavras que NUNCA usa (red flags de "não sou eu")

ESTRUTURA TÍPICA DO CLIENTE:
├── Como começa: pergunta? dado? história? afirmação direta?
├── Como desenvolve: lista? storytelling? argumento linear?
└── Como fecha: pergunta reflexiva? CTA? resumo? abertura para debate?

PERSONALIDADE — ESCALA DE POSICIONAMENTO:
├── Formal ←————————→ Informal
├── Humorístico ←————→ Sério
├── Opinativo ←————→ Neutro
└── Emojis: frequente / moderado / raro / nunca

TOP HOOKS DO CLIENTE:
Primeiras linhas dos 5 melhores posts — identifique o padrão.
(pergunta? número? história? contraintuitivo?)

ANTI-PATTERNS:
├── O que esse cliente NUNCA faria
├── Formatos que evita instintivamente
└── Expressões que soariam estranhas na voz dele

DO vs DON'T (exemplos reais):
Frases que são ele vs frases que não são ele.
Quanto mais específico, mais fácil de escalar o tom.
```

### Tom de Voz por Temperatura de Audiência

```
AUDIÊNCIA FRIA (nunca ouviu falar):
Tom: educativo, empático, sem pressão de venda
Foco: dor ou desejo — nunca o produto primeiro
Evite: jargão interno, nome da marca como solução, preço

AUDIÊNCIA MORNA (engajou, conhece você):
Tom: mais direto — já existe confiança parcial
Foco: diferencial, mecanismo, prova social específica
Evite: começar do zero como se não se conhecessem

AUDIÊNCIA QUENTE (lead, visitou LP, quase comprou):
Tom: direto, sem rodeio, remove fricção
Foco: objeções finais, urgência real, garantia
Evite: mais conteúdo educativo — eles já sabem, querem decidir
```

## Consistência de Mensagem — Do Criativo à Venda

O erro mais caro do marketing digital:
Anúncio promete X → LP fala sobre Y → produto entrega Z.
Cada quebra de alinhamento é conversão perdida.

```
TESTE DE ALINHAMENTO (aplique sempre):
1. Headline do anúncio e headline da LP usam as mesmas palavras-chave?
2. O avatar que o anúncio atrai é o mesmo que a LP converte?
3. O nível de consciência presumido é consistente entre anúncio e LP?
4. O CTA do anúncio e o próximo passo na LP são coerentes?

Qualquer "não" = vazamento de conversão nessa junção.
```

## Referências Clássicas — O Que Aplicar

### Gary Halbert
- **Lead de carta:** começa com o avatar, não com o produto
- **Especificidade extrema:** um número preciso vale mais que qualquer adjetivo
- **The boron letters:** escreva como se estivesse falando com uma pessoa que você conhece

### Eugene Schwartz
- **Níveis de consciência:** a copy deve encontrar o leitor onde ele está, não onde você quer que ele esteja
- **Força de desejo:** você não cria desejo — você o dirige e canaliza
- **Mecanismo único:** o produto não é o herói, o mecanismo que o diferencia é

### David Ogilvy
- **Headline:** 80 centavos do dólar estão na headline
- **Especificidade:** "A 60 miles an hour the loudest noise in this Rolls-Royce comes from the electric clock"
- **Não subestime o leitor:** o consumidor não é idiota — ele é seu cônjuge
