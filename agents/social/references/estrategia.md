# Estratégia de Conteúdo — Referência Sênior

Estratégia de conteúdo não é calendário. É a lógica por trás de cada peça:
quem vai ver, o que vai sentir, o que vai fazer depois.
Sem isso, você tem produção. Não estratégia.

---

## Avatar com Profundidade Real

Nunca use "público-alvo de 25-35 anos interessado em [nicho]".

**Formato de avatar útil:**
```
NOME FICTÍCIO: Marina
IDADE E CONTEXTO: 31 anos, empreendedora solo, atende clientes de design

SITUAÇÃO ATUAL:
├── Acorda às 3h preocupada com o fluxo de caixa
├── Trabalha 10h/dia mas sente que não avança
└── Tem clientes, mas não tem previsibilidade de receita

JÁ TENTOU:
├── 3 cursos de "como precificar"
├── Planilha de controle financeiro que abandonou em 2 semanas
└── Aumentou preço uma vez e perdeu cliente

DESEJO REAL (não o declarado):
Não quer só ganhar mais dinheiro. Quer parar de aceitar projeto ruim
porque "precisa" do dinheiro. Quer poder recusar.

O QUE TEM VERGONHA DE ADMITIR:
Sente que o problema é ela — que não é boa gestora, não é disciplinada.

LINGUAGEM QUE USA:
"Tô sobrando no mês mas não sei pra onde foi" /
"Odeio falar de dinheiro" / "Me sinto mal cobrando mais"
```

Esse nível de especificidade é o que separa copy que converte de copy que informa.

## Pilares de Conteúdo — Definição com Propósito

Cada pilar tem uma função específica no relacionamento com o avatar:

```
PILAR DA DOR
Função: fazer o avatar se reconhecer, criar senso de "esse conteúdo é pra mim"
Exemplos: erros comuns do nicho, situações que o avatar vive, frustrações reais
Risco se exagerado: audiência só de pessoas com problema, sem compradores

PILAR DE AUTORIDADE
Função: provar que você entende mais que ninguém sobre aquele assunto
Exemplos: dados, pesquisas, cases, análises profundas, bastidores de processo
Risco se exagerado: distante demais, parece inacessível

PILAR DE TRANSFORMAÇÃO
Função: provar que o resultado é real e possível para pessoas como o avatar
Exemplos: antes/depois, casos de clientes, resultados documentados
Risco se exagerado: parece só marketing, perde autenticidade

PILAR DE CRENÇA
Função: mudar o que o avatar acredita sobre o problema ou a solução
Exemplos: mitos do nicho desmontados, contraintuitivos, reposicionamento
Risco se exagerado: muito polarizante, pode alienar audiência

PILAR DE CONEXÃO
Função: humanizar, criar identificação, construir relacionamento
Exemplos: histórias pessoais, bastidores, opiniões, vulnerabilidade calibrada
Risco se exagerado: parece diário pessoal, perde autoridade
```

## Ângulo vs Tema — A Diferença que Muda Tudo

```
TEMA:   "Produtividade"
ÂNGULO: "Por que acordar às 5h é o conselho mais perigoso
         para quem tem filho pequeno"

TEMA:   "Tráfego pago"
ÂNGULO: "O criativo que mais vende raramente é o mais bonito —
         e os dados de 847 campanhas provam isso"

TEMA:   "Copy"
ÂNGULO: "Parei de usar AIDA por 30 dias. O que aconteceu com
         minhas conversões me surpreendeu"
```

**Regra:** O tema é a categoria. O ângulo é o ponto de vista específico
que faz aquele conteúdo diferente de tudo que já existe sobre o tema.
Conteúdo sem ângulo é commodity.

## Calendário Editorial — Como Estruturar

```
FREQUÊNCIA POR PLATAFORMA (referência):
├── Reels/TikTok: 3-5x/semana (volume alto, cadência rápida)
├── Feed estático/carrossel: 3-4x/semana
├── Stories: diário (bastidor, engajamento, humanização)
├── YouTube: 1x/semana (long-form) ou 2x (shorts + long)
├── LinkedIn: 3-4x/semana
├── Newsletter: 1-2x/semana (consistência > frequência)
└── X/Twitter: 1-3x/dia (threads 2-3x/semana)

DISTRIBUIÇÃO SEMANAL EQUILIBRADA:
Segunda:   Pilar de Autoridade (início da semana, tom mais sério)
Terça:     Pilar da Dor (identifica, gera engajamento)
Quarta:    Pilar de Transformação (prova social, resultado)
Quinta:    Pilar de Crença (contraintuitivo, gera discussão)
Sexta:     Pilar de Conexão (humanização, fim de semana)

MIX DE TEMPERATURA:
├── 60% topo de funil (alcance, novos seguidores, descoberta)
├── 30% meio de funil (engajamento, construção de confiança)
└── 10% fundo de funil (conversão, oferta, CTA direto)
```

## Content Waterfall — 1 Conteúdo Base → 8+ Formatos

```
CONTEÚDO BASE (vídeo longo, podcast, live, entrevista)
         ↓
EXTRAIA os 5-7 insights principais
         ↓
GERE EM PARALELO:
├── 1 Post LinkedIn (1200-1500 chars, gancho forte)
├── 1 Thread X/Twitter (5-8 tweets conectados)
├── 1 Roteiro Reels/TikTok (30-60s com o insight mais forte)
├── 1 Carrossel Instagram (7-9 slides)
├── 1 Bloco de Newsletter (300-500 palavras)
├── 1 Sequência de Stories (5-7 stories)
└── 3 Tweets avulsos (insights isolados, abordagens diferentes)

REGRA DO WATERFALL:
Formato nativo por plataforma — não é copiar o mesmo texto em todo lugar.
O mesmo insight, adaptado à linguagem e ritmo de cada canal.
```

## SOUL.md — Tom de Voz do Cliente

Documento que Luna cria e mantém por cliente.
Deve ser lido antes de criar qualquer conteúdo para aquele cliente.

```
SEÇÕES OBRIGATÓRIAS:

VOCABULÁRIO FREQUENTE:
├── Top 20 palavras/expressões que o cliente repete
├── Gírias e bordões que identificam o estilo
└── Palavras que NUNCA usa (red flags de "não sou eu")

ESTRUTURA TÍPICA:
├── Como começa (pergunta? dado? história? afirmação?)
├── Como desenvolve (storytelling? lista? argumento?)
└── Como fecha (pergunta? CTA? reflexão? resumo?)

PERSONALIDADE:
├── Formal ←→ Informal (onde se posiciona)
├── Humorístico ←→ Sério
├── Opinativo ←→ Neutro
└── Emojis: frequente / moderado / raro / nunca

GANCHOS QUE FUNCIONAM:
Top 5 primeiras linhas dos melhores posts do cliente.
Padrão identificado: pergunta? dado chocante? história pessoal?

ANTI-PATTERNS (o que não é esse cliente):
├── O que ele NUNCA faria
├── Formatos que evita
└── Expressões que soariam estranhas na voz dele

DO vs DON'T:
Exemplos reais — frases que são ele vs frases que não são.
```

## Análise de Concorrentes — Framework

```
Para cada concorrente relevante, mapeie:

POSICIONAMENTO:
├── Promessa principal de comunicação
├── Avatar explícito
└── Diferencial declarado

CONTEÚDO:
├── Formatos mais usados
├── Temas recorrentes
├── Tom de voz (formal/informal, educativo/entretenimento)
└── Posts com maior engajamento — por quê funcionaram?

GAPS E OPORTUNIDADES:
├── Temas que ninguém está cobrindo bem
├── Ângulos inexplorados no nicho
├── Formato que tem demanda mas pouca produção
└── Audiência que está sendo negligenciada

CONCLUSÃO:
Onde existe espaço que o cliente pode ocupar
e defender com credibilidade?
```
