# CREATIVE-BRIEF — GABI · Sub-skill
# Wolf Agency | Versão: 2.0

## TRIGGER
brief de criativo, criar brief, novo criativo, brief para design, brief para vídeo, o que criar de anúncio

## PROTOCOLO
1. Puxar todos os criativos ativos do cliente com dados dos últimos 30 dias (mínimo 500 impressões cada)
2. Selecionar os 3 criativos com melhor ROAS (ou menor CPA se e-commerce não for o objetivo)
3. Para cada um dos 3, documentar: formato (estático/vídeo/carrossel), ângulo de comunicação, CTA usado, público-alvo do adset, duração (se vídeo), elementos visuais principais
4. Identificar padrões comuns entre os 3 top performers: ângulo dominante, formato preferido, estilo visual, tipo de CTA
5. Cruzar com dados de público que mais converte (idade, gênero, placement)
6. Definir objetivo do novo criativo: aquisição fria / retargeting / upsell
7. Estruturar o brief com todos os campos do template Wolf
8. Incluir referências visuais (descrever ou linkar criativos de referência)
9. Definir formatos requeridos por placement: Feed 1:1, Stories 9:16, Reels 9:16, Banner Display
10. Registrar brief criado em activity.log vinculado ao cliente

## OUTPUT
```
CREATIVE BRIEF — [CLIENTE] — [DATA]

OBJETIVO: [aquisição/retargeting/upsell]
PÚBLICO: [descrição detalhada]
ÂNGULO PRINCIPAL: [baseado nos top performers]
MENSAGEM CENTRAL: [1 frase]
PROVA SOCIAL/ARGUMENTO: [dado, depoimento, resultado]
CTA: [exato]
REFERÊNCIAS: [criativos internos de referência]
FORMATOS: Feed 1:1 | Stories 9:16 | Reels 9:16
PRAZO DE ENTREGA: [data]
OBSERVAÇÕES: [restrições de marca, compliance]
```

## NUNCA
- Nunca criar brief sem analisar ao menos 3 criativos com dados reais do cliente
- Nunca recomendar ângulo sem embasamento em dados de performance
- Nunca omitir restrições de compliance da marca ou setor
