## BASE DE CONHECIMENTO NEWTON

### Teste Científico

**Template de hipótese obrigatório:**
```
SE [ação específica]
ENTÃO [resultado esperado com % estimada]
PORQUE [evidência nos dados que justifica]
VALIDO EM [período de observação]
```

**Hierarquia de testes (por impacto):**
```
1. GANCHO (3 primeiros segundos) — 70% do resultado
2. ÂNGULO — abordagem do problema/solução
3. FORMATO — vídeo, carrossel, imagem
4. CTA — botão, texto, posição
5. COPY — legenda, extensão, tom
```

**Regras de validação:**
| Elemento | Mínimo | Decisão |
|----------|--------|---------|
| Impressões | 3.000 | Significância estatística |
| Cliques | 30 | CTR confiável |
| Gasto | 1x CPA meta | Validação de custo |
| ROAS | 1.5x ticket médio | Validação de retorno |
| Frequência | >3.0 em frio | Sinal de saturação |

### CPA — Redução e Diagnóstico

```
CPA Plataforma = O que Ads Manager mostra
CPA Global     = (Gasto Ads + Custos) / Vendas reais
→ SEMPRE usar CPA Global para decisões de escala.
```

**Dentro da meta:** melhorar CTR (gancho), cortar segmentos CPA 20% acima, testar bundles
**Fora da meta:**

| Sintoma | Causa | Ação |
|---------|-------|------|
| CPM > R$40 | Saturação | Mudar targeting |
| CTR < 0.8% | Gancho fraco | 3 hooks novos |
| CPC > R$5 | Relevância baixa | Alinhar criativo-audiência |
| Conv. baixa | Landing/Oferta | Revisar funil |

**Árvore de diagnóstico:**
```
CPA ALTO (> meta + 25%)?
├── CTR baixo? → CRIATIVO
├── CTR ok + conversão baixa? → LANDING PAGE
├── CPM alto + frequência > 3? → SATURAÇÃO
├── CPM alto + frequência ok? → LEILÃO (horários, placements, bid)
└── Tudo ok mas volume baixo? → BUDGET (propor escala)
```

### Escala — 3 Caminhos

**VERTICAL:** +20% budget a cada 3-5 dias (risco: saturação)
**HORIZONTAL:** novos públicos, interests, lookalikes
**CRIATIVO:** variações do que validou, mesmo gancho, execuções diferentes

```
NUNCA aumentar mais que 20% do budget por dia.
Exceção: ROAS > 3x + CPA < 50% da meta

Sinais de saturação (pausar se 2+):
- Frequência > 2.5 em 7 dias
- CPM subindo > 30% em 3 dias
- CTR caindo > 20% em 3 dias
- CPA subindo > 20% em 3 dias
```

### Elementos de Criativo

**Ganchos — tipos por sofisticação:**
| Nível | Tipo | Exemplo |
|-------|------|---------|
| 1 | Novidade | "Novo método descoberto..." |
| 2 | Mecanismo | "A técnica do [nome]" |
| 3 | Resultado | "Como perder 5kg em 7 dias" |
| 4 | Contrarian | "Pare de fazer cardio" |
| 5 | Insider | "O que treinadores não contam" |

**Ângulos** (1 criativo = 1 ângulo):
Emocional | Racional | Social | Contrarian

**Formatos:**
| Formato | Objetivo | Métrica-chave |
|---------|----------|---------------|
| Vídeo 15-30s | Frio | Thumbnail CTR |
| Vídeo 60s+ | Quente | Retenção 50% |
| Carrossel | Educação | Taxa de swipe |
| Imagem | Remarketing | CTR |
| Stories/Reels | Engajamento | Taxa de toque |

**Specs técnicas Meta Ads:**
```
FEED: 1080x1080 (1:1) ou 1200x628 (1.91:1) · Texto até 125 chars · Headline até 40
STORIES/REELS: 9:16 · 1080x1920 · 15-30s · Área segura 250px topo/base
CARROSSEL: 2-10 cards · 1:1 ou 4:5 · headline por card (40 chars)
```

### Psicologia — Schwartz + Temperatura

| Nível | Estado | Estratégia |
|-------|--------|------------|
| 1-2 (Frio) | Inconsciente/Consciente do problema | Educação, agitação |
| 3 (Morno) | Consciente da solução | Solução + prova |
| 4-5 (Quente) | Consciente do produto/Desejo | Oferta + urgência |

### Automação

**PAUSAR:** CPA > 1.5x meta 2d | ROAS < breakeven 48h | Frequência > 3.0 7d
**AUMENTAR:** ROAS > 3x 3d | CPA < 70% meta 5d
**NOTIFICAR:** Campanha pausada por regra | CPA +30% 24h | Criativo sem impressão 6h

**Post ID Strategy:** criar post orgânico → pegar ID → usar em todos os anúncios (consolida engajamento)

### ROTINA-02 · Criação de Campanha

**Briefing obrigatório:**
```
□ Produto/serviço · Ticket médio · Margem líquida · Sazonalidade
□ Objetivo (Conversão/Lead/Alcance) · Meta CPA/CPL · Meta ROAS · Volume mínimo/mês
□ ICP (idade, gênero, localização) · Base para lookalike? · Tráfego para retargeting?
□ Quem produz criativos? · Aprovação prévia? · Formatos disponíveis? · Restrições?
```

**Estrutura padrão:**
```
CAMPANHA (Vendas/Leads)
├── TOPO — Frio (40-50% budget) · Interesse/Broad · 3-4 criativos
├── MEIO — Morno (25-30%) · Engajamento 30/60d + Visitantes LP · Prova social
└── FUNDO — Quente (20-30%) · Retargeting <30d + Carrinho · Oferta direta
```

**Checklist pré-lançamento:**
```
[ ] Pixel + CAPI disparando · Evento conversão testado
[ ] UTMs: ?utm_source=meta&utm_medium=cpc&utm_campaign=[NOME]&utm_content=[CONJUNTO]
[ ] Público > 500k · Sobreposição < 30% · Regras automáticas ativas
[ ] Mín. 3 criativos/conjunto · Specs corretas · Link testado mobile
[ ] Naming convention aplicado · Aprovação cliente registrada
```

**Nomenclatura:**
```
CAMPANHA:  [CLIENTE]_[OBJETIVO]_[PERÍODO]_[VERSÃO]    Ex: WOLF_LEADS_MAR26_V1
CONJUNTO:  [TEMP]_[PUBLICO-TIPO]_[BUDGET-DIARIO]      Ex: FRIO_INTERESSE-MKT_R100
ANÚNCIO:   [FORMATO]_[ANGULO]_[VERSÃO]                Ex: VIDEO_PROVA-SOCIAL_V1
```

### ROTINA-03 · Análise Semanal (segunda até 10h, 45-90 min)
```
1. Performance semana vs anterior · Variação >20% → investigar
2. Top 3 e bottom 3 criativos por CTR e CPA · Frequência >2.5 → substituir
3. Audiências: qual segmento converte melhor? Sobreposição? Lookalike?
4. Placements: Feed vs Stories vs Reels — realocar para vencedores
5. Plano semana seguinte: 3-5 otimizações com hipóteses documentadas
```

### ROTINA-04 · Janelas de Otimização
```
SEMANA 1-2 (Aprendizado — NÃO MEXER):
→ Pausar só reprovados ou zero entrega 48h · Monitorar CPM, alcance

SEMANA 2-4 (Primeira Otimização):
→ Pausar CPA > 200% meta · Pausar CTR 50% abaixo da média
→ Escalar +20-30% conjuntos com CPA dentro da meta

MENSAL (Reestruturação):
→ Atualizar lookalikes 30d · Arquivar campanhas >90d · Revisar metas com cliente
```

**Semanal (WhatsApp, segunda até 10h):**

- **Positivo:** número exato + o que contribuiu + próximo passo

**KPIs Core:**

| SaaS / B2B | 0.5-1% | R$30-60 | R$30-100 | — |

## PRINCÍPIOS INEGOCIÁVEIS

1. **Dado antes de opinião** — "Acho que" não vale, métrica vale
2. **Teste antes de escala** — Nunca escalar sem validar
3. **CPA antes de ego** — Se não performa, pausa, não justifica
4. **Uma variável por teste** — Testar múltiplas = não saber o que funcionou
5. **Documentar tudo** — Teste sem registro é teste perdido
6. **Frequência é sagrada** — >3.0 em frio = pausa imediata
7. **Regra dos 20%** — Paciência na escala
8. **Budget é do cliente** — Jamais gastar além do autorizado
9. **Sem pixel, sem campanha** — Rastreamento é pré-requisito absoluto
10. **Alerta crítico não dorme** — Resposta em até 2h

## GLOSSÁRIO OPERACIONAL

| Termo | Definição |
|-------|-----------|
| CPA meta | (Ticket × Margem Bruta) ÷ 3 |
| ROAS breakeven | 1 ÷ Margem Bruta |

| Temperatura | Frio: nunca ouviu · Morno: engajou · Quente: visitou/lead/comprador |
| CAC sustentável | CAC < LTV ÷ 3 |
| Sobreposição | >30% entre conjuntos = competição interna no leilão |
| Aprendizado | <50 conversões/semana = não edite o conjunto |
| Lookalike | Público similar aos clientes |
| Post ID | ID único de post — consolida engajamento entre anúncios |
| Broad | Segmentação aberta sem interesse definido |
