## Checklist Técnico (Todos os Formatos)

### Vídeo
- [ ] Resolução correta (1080x1920 ou 1920x1080)
- [ ] Frame rate consistente (30fps ou 60fps)
- [ ] Codec adequado (H.264 para web)
- [ ] Bitrate suficiente (10+ Mbps)
- [ ] Sem artefatos de compressão
- [ ] Color grading consistente
- [ ] Sem frames pretos no início/fim

### Áudio
- [ ] Nível de voz: -12dB a -6dB
- [ ] Música: -18dB a -24dB (abaixo da voz)
- [ ] SFX: -12dB a -6dB (pontuais)
- [ ] Sem clipping
- [ ] Sem ruído de fundo excessivo
- [ ] Compressão aplicada na voz
- [ ] De-esser se necessário

### Legendas
- [ ] Fonte legível
- [ ] Tamanho adequado
- [ ] Contraste suficiente
- [ ] Posição correta (safe area)
- [ ] Sem erros de ortografia
- [ ] Sincronizadas com fala
- [ ] Duração mínima: 1.5s por tela

### Entrega
- [ ] Formato correto (.mp4)
- [ ] Tamanho adequado (< 100MB)
- [ ] Nome do arquivo padronizado
- [ ] Thumbnail incluída
- [ ] Versões alternativas (se solicitado)

## Processo de Revisão

### Etapa 1: Auto-QC (Editor)
- [ ] Aplicar scorecard no próprio trabalho
- [ ] Corrigir itens < 7 antes de enviar
- [ ] Verificar checklist técnico

### Etapa 2: Revisão Interna
- [ ] Segunda pessoa aplica scorecard
- [ ] Feedback estruturado (números específicos)
- [ ] Ajustes realizados

### Etapa 3: Aprovação Cliente
- [ ] Versão de review enviada
- [ ] Feedback coletado
- [ ] Ajustes finais
- [ ] Aprovação formal
