# Reels/Stories Playbook — Video Editor Pro

> SOP completo para short-form de alta performance

---

## Fórmula de 45 Segundos

```
0–2s   → HOOK (pára o scroll)
2–6s   → CONTEXTO (qual é o problema)
6–20s  → PROVA/BENEFÍCIO (resultado real)
20–45s → 2–3 PONTOS (pattern interrupts)
Final  → CTA CLARO (+ loop)
```

---

## Hook em 0–2 Segundos

### Técnicas de Hook Visual

| Técnica | Exemplo | Uso |
|---------|---------|-----|
| **Resultado chocante** | "R$ 0 → R$ 50k" | Produtos financeiros |
| **Transformação** | Before/after rápido | Fitness, beleza |
| **Reação genuína** | Surpresa/espanto | UGC, reviews |
| **Texto grande** | "PARE DE FAZER ISSO" | Educação |
| **Movimento abrupto** | Zoom rápido, corte seco | Qualquer nicho |

### Hook Textual (Legendas)

```
❌ "Hoje vou falar sobre..."
✅ "3 erros que me custaram R$ 10 mil"

❌ "Você sabia que..."
✅ "NINGUÉM te conta isso sobre [tema]"

❌ "Dica importante:"
✅ "Faz assim e me agradece depois"
```

## Legibilidade Mobile

### Safe Areas

```
┌─────────────────────────────┐
│  ⚠️ EVITAR (status bar)     │  ← 10% topo
│                             │
│    CONTEÚDO SEGURO          │  ← 80% centro
│    (legendas aqui)          │
│                             │
│  ⚠️ EVITAR (navegação)      │  ← 10% base
└─────────────────────────────┘
```

### Especificações de Legenda

| Elemento | Especificação |
|----------|---------------|
| **Fonte** | Montserrat, Poppins, ou similar sans-serif |
| **Tamanho** | 60–80pt (proporcional ao vídeo) |
| **Linhas** | Máximo 2 |
| **Caracteres** | Máximo 40 por linha |
| **Cor** | Branco com stroke preto 4px |
| **Posição** | Centro inferior (safe area) |
| **Duração** | Mínimo 1.5s na tela |

### Modos de Legenda

**Word-by-word (Alta retenção):**
```
"Como" (0.3s) → "eu" (0.3s) → "fiz" (0.3s) → "50k" (0.5s)
```

**Blocos (Premium):**
```
"Como eu fiz 50k em 30 dias" (2s)
```

**Híbrido (Default):**
```
Palavras-chave word-by-word + frases em bloco
```

## Pattern Interrupts

### Tipos de Interrupt

| Tipo | Frequência | Exemplo |
|------|------------|---------|
| **Zoom** | A cada 5–8s | 100% → 110% suave |
| **Crop** | A cada 10s | Reenquadramento |
| **SFX** | Pontual | "Whoosh", "ding", "pop" |
| **Texto popup** | 2–3 por vídeo | Destaque de palavra |
| **Mudança de cena** | A cada 15s | Cut seco |

### Cuidados

- ❌ Não exagerar (máx 1 interrupt a cada 5s)
- ❌ Não poluir visual
- ✅ Cada interrupt deve ter função (atenção ou clareza)

## Loopability

### Técnicas de Loop

| Técnica | Como Fazer |
|---------|------------|
| **Match cut** | Último frame = primeiro frame |
| **Transição suave** | Fade ou dissolve no final |
| **Repetição musical** | Música termina onde começa |
| **Hook no final** | Último segundo lembra o primeiro |

### Indicadores de Loop Bem-sucedido

- Usuário assiste 2x+ sem perceber
- Taxa de conclusão > 100%
- Comentários "assistiu 5x e não viu"

## Densidade de Cortes

### Por Formato

| Formato | Cortes/Min | Duração Média do Cut |
|---------|------------|----------------------|
| Reels | 15–25 | 2–4s |
| Stories | 10–15 | 3–5s |
| TikTok | 20–30 | 1.5–3s |

### Quando Cortar

- Pausa na fala (> 0.5s)
- Mudança de ideia
- Transição de emoção
- Antes de um reveal
- Depois de um punchline

## Checklist de Qualidade Short-form

### Pré-Produção
- [ ] Hook definido (visual + textual)
- [ ] Roteiro de 45s ou menos
- [ ] 2–3 pattern interrupts planejados
- [ ] CTA claro
- [ ] Música selecionada (ritmo adequado)

### Edição
- [ ] Hook nos primeiros 2s
- [ ] Legendas em todo o vídeo
- [ ] Safe areas respeitadas
- [ ] Cortes dinâmicos
- [ ] SFX pontuais
- [ ] Loopability considerada

### Pós
- [ ] Primeiro frame forte
- [ ] Último frame leva ao replay
- [ ] Export em 1080x1920
- [ ] Tamanho < 50MB (Instagram)
- [ ] Thumbnail atrativa
